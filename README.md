## PCB defect detection with YOLOv8

simple implementation of YOLOv8 to detect different defects on PCBs implemented on Hugging face with Gradio

0 - missing hole </br>
1 - mouse bite </br>
2 - open circuit </br>
3 - short </br>
4 - spur </br>
5 - spurious copper </br>
